# OList-Marketing-and-Retail-Analytics
End-to-end Marketing and Retail Analytics project including EDA, Pareto Analysis, and Market Basket Analysis using Python and Tableau.

📊 OList Marketing and Retail Analytics

🔹 Project Overview

This project analyzes the OList Retail dataset to uncover insights about product performance, revenue distribution, and customer purchasing patterns.

🔹 The analysis includes:

Exploratory Data Analysis (EDA)

Pareto Analysis (Revenue & Quantity)

Running Totals Analysis

Top 20 Product Analysis

Market Basket Analysis

Tableau Dashboard Visualization

🔹 Tools & Technologies Used

Python (Pandas, NumPy)

Matplotlib & Seaborn

Jupyter Notebook

Tableau

Excel

🔹 Key Business Insights

✅ Toys category contributes ~76% of total revenue

✅ Toys, Health Beauty, and Watches Gifts generate 80%+ revenue (Pareto principle)

✅ Market Basket Analysis shows strong association between:

Toys

Bed Bath Table

Furniture Decor

Computers Accessories

Health Beauty
